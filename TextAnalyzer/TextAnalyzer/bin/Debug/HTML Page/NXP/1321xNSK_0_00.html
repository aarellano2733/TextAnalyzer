Overview

Based on NXP ® Semiconductors's 2nd generation of Zigbee™ Compliant platforms, the 1321xNSK (Network Starter Kit) provides a solid basis for demonstrating and developing more complex networks, including star and mesh networks.

Archived content is no longer updated and is made available for historical reference only.

Features

  • Two 1321x-SRB (Sensor Reference Board) boards
    • 2.4 GHz wireless nodes compatible with the IEEE 802.15.4 standard
    • MC13213 2.4 GHz SiP
    • MMA7260Q 3-axis acceleration sensor and temperature sensor for easy demonstration of wireless data collection
    • Printed F antenna
    • Onboard expansion capabilities for external application-specific development activities
    • Programmable 60K flash with 4K of RAM
    • Onboard BDM port for MCU Flash reprogramming and in-circuit hardware debugging
    • USB port to interface with PC
    • LEDs and switches for demonstration, monitoring and control
    • Connections for battery or external power supply
  • One 1321x-NCB (Network Coordinator Board).
    • 2.4 GHz wireless nodes compatible with the IEEE 802.15.4 standard
    • Programmable 2-line LCD for displaying messages
    • MC13213 2.4 GHz SiP
    • Printed F antenna
    • Onboard expansion capabilities for external application-specific development activities
    • Programmable 60K flash with 4K of RAM
    • Onboard BDM port for MCU Flash reprogramming and in-circuit hardware debugging
    • USB and serial port to interface with PC
    • LEDs and switches for demonstration, monitoring and control
    • Connections for battery or external power supply
  • Supports SMAC, IEEE 802.15.4 and ZigBee™
  • Includes CodeWarrior ® ™ Development Studio for HCS08 Special Edition
  • Power Adapters, Batteries and Cables
  • USB Multilink BDM Programmer/Debugger(1321xNSK-BDM only)

Note: Requires an HCS08 BDM Multilink Programmer for debug and program capability
